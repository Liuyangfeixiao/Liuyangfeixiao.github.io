LeetCode 316 去除重复字母

题目描述：

给你一个字符串 s ，请你去除字符串中重复的字母，使得每个字母只出现一次。需保证 返回结果的字典序最小（要求不能打乱其他字符的相对位置）。

提示：

  • 1 <= s.length <= 104
  • s 由小写英文字母组成

分析：

要去除字符串中的重复字符，

  1. 我们需要依次遍历字符串的每个字符，用一个单调栈保留处理过的字符
  2. 若当前遍历的字符在栈中，则直接抛弃
  3. 否则将当前遍历的字符与栈顶的字符比较，若是大于，则判断栈顶字符是否可以弹出
  4. 若是栈顶的字符在后续字符中无重复，则不可出栈，进行下一次遍历
