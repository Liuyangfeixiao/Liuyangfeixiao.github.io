二分图问题

二部图由叫做二分图，是图论的一种特殊模型。简单来说，如果图中的点可以分为两组，图中的每一条边都是连接组间的边，这就是一个二分图。

二分图有一个重要定理：G 为二分图的充要条件是 G 中的每一个环的长度都是偶数

于是我们可以用染色法来判别二分图

染色法判别二分图

通过上面的定理。我们可以逐个给节点染色，每个节点的颜色与相邻的节点不同，如果相邻节点已经被染色且颜色与本节点相同，则存在偶数环，不是二分图。

二分图的最大匹配

匈牙利算法：先来先到，能让就让，联通边数只会增加，不会减少。

匈牙利算法的应用

棋盘覆盖

给定一个N行N列的棋盘，已知某些格子禁止放置。

求最多能往棋盘上放多少块的长度为2、宽度为1的骨牌，骨牌的边界与格线重合（骨牌占用两个格子），并且任意两张骨牌都不重叠。

输入格式

第一行包含两个整数N和t，其中t为禁止放置的格子的数量。

接下来t行每行包含两个整数x和y，表示位于第x行第y列的格子禁止放置，行列数从1开始。

输出格式

输出一个整数，表示结果。

数据范围

1 ≤ N ≤ 1001 ≤ N ≤100,
0 ≤ t ≤ 1000 ≤ t ≤ 100

分析：

首先我们可以将此题转换， 将棋盘格子看作是一个节点，相邻的节点之间有边连接，此题就可以转换成最多可以从图中取出多少不含相同节点的边，同时也不含禁止节点的边。

我们可以将行 + 列为奇数的格子叫做奇数格，行 + 列为偶数的格子为偶数格，可以发现奇数格的周围必然是偶数格，也就是说图中的边都连接两个节点集合，于是题目就转化为二分图求最大匹配问题。
